## Description
Web-based search engine capable of indexing 10,000+ documents and providing search results based on queries. This search engine was created using Python, alongside numerous libraries, to access a plethora of documents and their content. Large amounts of 
text is scraped and analyzed to pair keywords with documents. Term/Document scoring techniques are utilized for optimizing and narrying down search results. An index is created to store a json dictionary of words and documents for later use with the search 
engine. <br/>
[Source](https://github.com/jdinh-782/information-retrieval-search-engine)
